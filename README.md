# 🌍 World Atlas

React-приложение для просмотра информации о странах мира с поиском, фильтрацией по региону и детальной карточкой.

## Стек

- **React 18** + **TypeScript**
- **Vite** — сборщик
- **Sass + SCSS-модули** — стилизация
- **fetch** — запросы к API
- API: [restcountries.com](https://restcountries.com/v3.1/all)

## Функциональность

- 📋 Список всех стран в виде карточек (флаг + название)
- 🔍 Поиск по названию страны (без учёта регистра)
- 🌐 Фильтрация по региону (Africa, Americas, Asia, Europe, Oceania)
- 📄 Детальная информация при клике: флаг, название, столица, регион, население
- 📱 Адаптивный дизайн (mobile-first)

## Локальный запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр собранного проекта
npm run preview
```

Приложение откроется на `http://localhost:5173`
